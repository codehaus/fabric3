Roadmap | Docs | Downloads

This roadmap is organized by milestone dates on which we intend (with no guarantees) to release certain groups of modules and the enhancements planned for each module.

March 2008 - Vegas

  • CDI for Resources
  • Updating wires as new components are activated

Unspecified future time

  • fabric3-spi
    • Support SCDL parsing based on XML type (including xsi:type)
    • Simplify ComponentType by removing type parameters
  • fabric3-pojo
    • Refactor PojoComponentType to replace JavaMapped* with InjectionSiteMapping
  • fabric3-transform
    • Support for data types with type parameters
  • fabric3-fabric
    • Allocate components based on features advertised by runtime
    • Provision features to runtimes as needed by components
  • fabric3-binding-burlap 0.2
    • Callbacks
    • Conversations
  • fabric3-binding-hessian 0.2
    • Callbacks
    • Conversations
  • fabric3-binding-jms 0.2
    • Default Data binding
    • Callbacks
    • Conversations
    • Activation Spec
    • JCA
  • fabric3-binding-ws 0.2
    • Callbacks
    • Conversations
  • General
    • Support for public and private endpoint addresses for NAT configs
    • RMI binding
    • JSON-RPC binding
    • Spring implementation
    • JDBC Integration
    • File drop binding
    • FTP binding
    • Support for JSR-223 based pluggable script implementations
    • Add transformer interceptor framework for bindings
    • Refactor JMS messaging service to use ActiveMQ network of brokers topology
    • Controller and participant runtime split
    • Automatically bind remote wires
