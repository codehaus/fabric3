The tutorials provide an introduction to Fabric3, including setting up a development environment, deploying an application to a Fabric3 server, and multi-VM service provisioning.

The tutorials consist of the following modules:

  • calculator - An introduction to Fabric3.
  • calculator-webclient - A web front-end to the calculator application.
  • loanapp - A complete application demonstrating Fabric3 service provisioning.
  • loanapp-client - A client application used to connect remotely to the loan processing application.

Setup

The tutorials are distributed as source in the Downloads Section and are built using Maven 2.0.7 or later. The tutorials also include pre-configured IDE project files for Eclipse and IDEA.

To setup a development environment you will need to perform the following steps:

  1. Download and install Maven 2.0.7 or later .
  2. Download and unpack the Fabric3 Tutorials distribution. Verify the distribution by building the samples using mvn -r clean install from the top-level directory. Note that modules may be built individually by executing mvn clean install from their directories.
  3. Download and unpack the Fabric3 Development Runtime
  4. Open and configure the tutorials project bundled with the tutorials distribution. Follow the specific setup instructions for IDEA or [Eclipse].
